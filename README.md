# Ngario

一個基於 Phaser 3 的 Agario 風格遊戲。

## 遊戲特點

- 完全離線運行
- 支持分裂和噴射質量
- 智能 AI 敵人
- 無限大地圖
- 網格背景
- 排行榜系統
- 歷史人物命名的敵人

## 控制方式

- 方向鍵：移動
- 空格鍵：分裂
- W 鍵：噴射質量

## 遊戲規則

- 吃綠色食物可以增加質量
- 吃比自己小的敵人可以獲得其質量
- 被比自己大的敵人吃掉會遊戲結束
- 可以分裂成多個碎片
- 可以噴射質量來加速

## 如何運行

1. 下載或克隆此倉庫
2. 打開 `index.html` 文件即可開始遊戲

## 技術細節

- 使用 Phaser 3 遊戲引擎
- 純 JavaScript 實現
- 無需安裝任何依賴
